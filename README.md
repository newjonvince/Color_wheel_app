# Fashion Color Wheel App

A mobile app that helps users match colors for fashion using the device's camera. The app allows users to capture colors, match them using different color schemes (complementary, monochromatic, analogous, triadic, or tetradic), and save their favorite outfit combinations.

## Features

- Camera integration for color capture
- Multiple color scheme options:
  - Complementary
  - Monochromatic
  - Analogous
  - Triadic
  - Tetradic
- Save and manage outfit combinations
- Front/back camera switching

## Installation

1. Make sure you have Node.js installed on your system
2. Install Expo CLI globally:
   ```bash
   npm install -g expo-cli
   ```
3. Install project dependencies:
   ```bash
   npm install
   ```

## Running the App

1. Start the development server:
   ```bash
   npm start
   ```
2. Install the Expo Go app on your mobile device
3. Scan the QR code shown in the terminal with your device's camera
4. The app will open in Expo Go

## Requirements

- Node.js
- Expo CLI
- iOS or Android device with Expo Go installed
